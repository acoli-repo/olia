Annotations (1)

  • comment: "The iterative aspect describes the occurrence of multiple events within a single time frame. This can be represented as multiple TSit spans within a given TT. This TT and the number of TSit spans must be bounded, since if it is infinite or unbounded, it gives rise to a habitual interpretation. The habitual aspect is essentially the iterative aspect, but with infinitely many multiple situation times within an unbounded topic time.
    (Sylak-Glassman 2016, p.15)" (en)
